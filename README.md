# Azure Villas

A luxury villa rental website for Azure Villas in Lefkada, Greece. Built with Next.js, TypeScript, and TailwindCSS.

## Features

- **Modern Design**: Elegant, contemporary design inspired by luxury villa aesthetics
- **Responsive**: Fully responsive layout for all devices
- **Image Gallery**: Interactive lightbox gallery for villa photos
- **Smooth Animations**: Hero slider with auto-play and smooth transitions
- **Contact Section**: Easy access to contact information

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

### Development

The development server will start at [http://localhost:3000](http://localhost:3000).
